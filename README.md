# 🌤️ Weather Forecast App

A modern, responsive, and feature-rich Weather Forecast Application that provides real-time weather updates, air quality information, hourly forecasts, 5-day predictions, weather alerts, and interactive temperature charts. Built with HTML, CSS, JavaScript, Tailwind CSS, Chart.js, and OpenWeatherMap API.

## 📖 Overview

Weather Forecast App is a modern weather dashboard that allows users to check real-time weather conditions for any city worldwide or automatically detect weather based on their current location. It also provides AQI information, weather alerts, hourly forecasts, 5-day forecasts, and graphical temperature analysis.

## 🚀 Getting Started

### Prerequisites

Before running this project, ensure you have:

- A modern web browser
- OpenWeatherMap API Key

---

## 🔑 Getting an API Key

1. Visit https://openweathermap.org
2. Create an account
3. Navigate to API Keys section
4. Generate a new API key
5. Copy your API key

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/AnshuRaj/Weather-Forecast-App.git
```

### Navigate to Project Folder

```bash
cd Weather-Forecast-App
```

### Configure API Key

Open `script.js`

Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with:

```javascript
const apiKey = "YOUR_ACTUAL_API_KEY";
```

---

## ▶️ Run the Application

Simply open:

```bash
index.html
```

or use VS Code Live Server Extension.

## 📡 API Endpoints Used

### Geocoding API

```http
https://api.openweathermap.org/geo/1.0/direct
```

### Current Weather API

```http
https://api.openweathermap.org/data/2.5/weather
```

### Forecast API

```http
https://api.openweathermap.org/data/2.5/forecast
```

### Air Pollution API

```http
https://api.openweathermap.org/data/2.5/air_pollution
```

### Weather Alerts API

```http
https://api.openweathermap.org/data/2.5/onecall
```
