# 🌤 Weather Forecast App

A modern and responsive Weather Forecast Application that provides real-time weather updates, AQI monitoring, hourly forecasts, 5-day predictions, weather alerts, and interactive temperature charts.

## 🚀 Features

### 🌍 Weather Information
- Real-time weather updates
- Current temperature
- Weather conditions and icons
- Feels-like temperature
- Min/Max temperature
- Humidity information
- Wind speed details

### 📍 Location Support
- Search weather by city name
- Automatic current location detection using Geolocation API

### 🌫 Air Quality Monitoring
- Live AQI (Air Quality Index)
- AQI status indicators:
  - Good
  - Fair
  - Moderate
  - Poor
  - Very Poor

### ⏰ Forecasts
- Next 24-hour weather forecast
- 5-day weather forecast

### 📈 Data Visualization
- Interactive temperature trend chart
- Forecast temperature analysis using Chart.js

### 🌅 Additional Information
- Sunrise time
- Sunset time
- Weather alerts and warnings

### 🎨 User Experience
- Fully responsive design
- Dark/Light mode toggle
- Animated weather background
- Smooth transitions and hover effects
- Loading indicators
- Error handling

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- Chart.js
- OpenWeatherMap API
